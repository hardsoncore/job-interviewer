<p class="info">
  <span class="accent">Псевдокласи</span> — це спеціальні селектори, які дозволяють стилізувати елементи залежно від їхнього стану, положення в структурі документа або інших умов. Вони починаються з двокрапки <code>:</code>.
</p>

<h2>1. Псевдокласи стану (Інтерактив)</h2>
<p>Ці селектори реагують на дії користувача:</p>

<ul>
    <li><code>:hover</code> — коли курсор миші знаходиться над елементом.</li>
    <li><code>:active</code> — момент кліка (утримання кнопки миші).</li>
    <li><code>:focus</code> — коли елемент вибрано (наприклад, через Tab або клік).</li>
    <li><code>:focus-visible</code> — спрацьовує тільки при навігації з клавіатури.</li>
</ul>

<h2>2. Структурні псевдокласи</h2>
<p>Допомагають стилізувати елементи залежно від їхнього положення в DOM:</p>

<table>
    <thead>
        <tr>
            <th>Псевдоклас</th>
            <th>Опис</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td><code>:first-child</code></td>
            <td>Перший дочірній елемент свого батька.</td>
        </tr>
        <tr>
            <td><code>:last-child</code></td>
            <td>Останній дочірній елемент свого батька.</td>
        </tr>
        <tr>
            <td><code>:nth-child(n)</code></td>
            <td>Вибір за номером або формулою (напр., 2n — парні).</td>
        </tr>
        <tr>
            <td><code>:only-child</code></td>
            <td>Якщо елемент — єдиний дочірній елемент.</td>
        </tr>
        <tr>
            <td><code>:not(selector)</code></td>
            <td>Виняток: вибирає все, крім зазначеного.</td>
        </tr>
    </tbody>
</table>

<h2>3. Просунуті селектори (Modern CSS)</h2>

<p><code>:has()</code> — «батьківський селектор». Дозволяє стилізувати батьківський елемент, якщо всередині нього є певний елемент.</p>

<p>Приклад: <code>div:has(img)</code> — вибере тільки ті <code>div</code>, всередині яких є картинка.</p>

<p>
  <code>:is()</code> — групування селекторів для чистоти коду.
  Він дозволяє групувати селектори замість того, щоб перераховувати довгі ланцюжки через кому.
</p>

<code class="code">
  /* Без :is() */
  .content h1,
  .content h2,
  .content h3,
  .section h1,
  .section h2,
  .section h3 {
    color: red;
  }

  /* З :is() */
  :is(.content, .section) :is(h1, h2, h3) {
    color: red;
  }
</code>

<p><code>:where()</code> — аналог <code>:is()</code>, але з нульовою специфічністю (легко перебивається іншими стилями).</p>

<h2>4. Псевдокласи форм</h2>

<p>Ці селектори застосовуються до елементів форм залежно від їхнього стану або атрибутів:</p>

<p><code>:disabled</code> — якщо поле заблоковано.</p>

<p><code>:required</code> — якщо у поля є атрибут обов'язкового заповнення.</p>

<p><code>:invalid</code> / <code>:valid</code> — автоматична перевірка (наприклад, якщо в <code>type="email"</code> введено щось інше).</p>
<p><code>:checked</code> — для вибраних чекбоксів та радіокнопок.</p>

<p><code>:placeholder-shown</code> — стиль поля, поки в ньому нічого не написано.</p>
