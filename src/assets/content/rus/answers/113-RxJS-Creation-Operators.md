<h3>of</h3>
<p>Оператор <code>of</code> используется для создания Observable из набора значений. Он принимает любое количество аргументов и возвращает Observable, который эмитирует эти значения по одному, а затем завершает поток.</p>

<code class="code">
  import { of } from 'rxjs';
  of(1, 2, 3).subscribe(value => console.log(value));
  // → 1, 2, 3, complete
</code>

<h3>from</h3>
<p>Оператор <code>from</code> используется для создания Observable из различных типов данных (массивов, промисов, итераторов и т.д.). Он преобразует эти данные в поток, эмитируя каждое значение по отдельности.</p>

<code class="code">
  import { from } from 'rxjs';
  from([10, 20, 30]).subscribe(value => console.log(value));
  // → 10, 20, 30, complete
</code>

<h3>interval</h3>
<p>Оператор <code>interval</code> создает Observable, который эмитирует последовательность чисел с определенным интервалом времени. Он принимает один аргумент - интервал в миллисекундах - и начинает эмитировать числа, начиная с 0, с заданным интервалом.</p>

<code class="code">
  import { interval } from 'rxjs';
  interval(1000).subscribe(value => console.log(value));
  // → 0, 1, 2, 3, ... (каждую секунду)
</code>

<h3>timer</h3>
<p>Оператор <code>timer</code> создает Observable, который эмитирует первое значение после определенной задержки, а затем продолжает эмитировать последовательность чисел с заданным интервалом. Он принимает два аргумента: первый - задержка перед первым эмитированием, второй - интервал между последующими эмитированиями.</p>

<code class="code">
  import { timer } from 'rxjs';
  timer(2000, 1000).subscribe(value => console.log(value));
  // → (через 2 секунды) 0, (через 3 секунды) 1, (через 4 секунды) 2, ...
</code>
