
<p>
  <span class="accent">Наследование в JavaScript</span> — это механизм, с помощью которого один объект может получать доступ
  к свойствам и методам другого объекта.
</p>

<p>
  Главная особенность JavaScript заключается в том, что он использует прототипное наследование
  (<span class="accent">prototypal inheritance</span>), в отличие от классического наследования (на основе классов),
  которое применяется в языках вроде Java или C++.
</p>

<p>
  В JavaScript практически всё является объектом. У каждого объекта есть скрытое внутреннее свойство <code>[[Prototype]]</code>
  (которое исторически доступно через геттер/сеттер <code>__proto__</code>).
  Оно ссылается на другой объект — его прототип.
</p>

<p>
  Когда вы пытаетесь прочитать свойство или вызвать метод у объекта, JavaScript сначала ищет это свойство или метод
  в самом объекте, а потом идет по цепочке прототипов <span class="accent">(Prototype Chain)</span>.
</p>
